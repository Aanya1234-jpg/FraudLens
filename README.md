# FraudLens AI: Institutional Risk Orchestration System 🛡️

**FraudLens** is an institution-grade fraud detection and risk management platform designed for modern banking security operations centers (SOC). It utilizes Machine Learning and Behavioral Analytics to assign a real-time **Risk Probability Index (RPI)** to financial transactions.

## 🚀 Core Features
- **Real-Time Surveillance:** Live transaction firehose with automated RPI scoring.
- **Forensic deep-dives:** Subject investigation module using **Explainable AI (XAI)**.
- **Risk Attribution:** SHAP-based waterfall plots to justify security decisions.
- **Dynamic Policy Management:** Sidebar-driven "Sensitivity Slider" to adjust risk appetite.
- **Localized Intelligence:** Geo-spatial threat map focused on the Ranchi/Jharkhand region.

## 🛠️ Technology Stack
- **Backend:** Python 3.12, Scikit-Learn (Random Forest), Imbalanced-Learn (SMOTE)
- **Frontend:** Streamlit with Glassmorphism UI design.
- **Explainability:** SHAP (Shapley Additive Explanations).
- **Visualization:** Plotly Express.
